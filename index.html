Genetic Transcription

Enter a DNA sequence consisting only of the characters A, C, G, T.
For example, if you enter "ACGT", the result will be "UGCA".

Palindrome

Enter a string to check if it is a palindrome.
For example, if you enter "racecar", the result will be "true".

Substring

Enter a string from which to extract a substring.
For example, if you enter "Hello, world!" without specifying start and length, the result will be "Hello, world!".
